# projeto-db-concecionaria-cruid

Para executar o projeto, siga os passos abaixo:
Extraia o arquivo .zip dentro do diretório htdocs do XAMPP
(ou dentro da pasta equivalente de outro servidor local, como o Wamp).
Abra o XAMPP Control Panel e inicie os seguintes serviços:
Apache
MySQL
Após iniciar os serviços, abra o navegador e digite:
Copy code

http://localhost/nome-da-pasta/index.php
Substitua nome-da-pasta pelo nome da pasta do projeto que você colocou dentro do htdocs.
